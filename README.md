
## PollView

[Website Link](https://dig4503-midterm.onrender.com/)

<img src="https://github.com/TybeeS/TybeeS_ClaudePortfolio/blob/main/img/pollview.jpg?raw=true" width="600">

PollView is a website that allows users can create polls on a variety of topics and submit these polls for others to vote on them. These polls can range from multiple choice to open-ended.


### Features:
- Easy creation of polls of various types, including multiple choice and open-ended responses.
- Generate link to poll for others to access and vote on.
- Login to save polls via user accounts.
- View dashboard of created polls with the results of various polls displayed as they occur.
- Search/filter polls by category (e.g., politics, entertainment) and creation/voting timeframe.

### Techstack: 
- Node.js
- CSS for styling
- Render for deployment
- Firebase for user authentication

This project was the first time I had ever used AI to develop a type of "web service." I have developed static sites before, but PollView required much more attention towards maintaining and receiving data. I learned how to be more involved in my conversations with Claude. Instead of just asking it to do certain things, I would look over its decision-making and make my own choices in the number of times I was presented with them. I already use Firebase for authentication, but I would also want to use it for storing polls as well. That way, polls would be able to persist across deploys instead of being wiped every time Render restarts the server.

## Dazzling Donuts

[Website Link](https://earnest-entremet-78f7e8.netlify.app/)

<img src="https://github.com/TybeeS/TybeeS_ClaudePortfolio/blob/main/img/dazzlingdonuts.jpg?raw=true" width="600">

Dazzling Donuts is a simple website mockup for a fictional donut shop. The site showcases the shop's menu, story, and     
contact information.

### Features:  
- A full menu of donuts with descriptions and pricing.
- Search and filter tool to sort items by type (e.g., glazed, chocolate, fruity)
- Learn about the shop's story and values on a dedicated About page.
- Contact page with location, hours, and a form to reach the shop directly.
- Responsive design for seamless browsing on both desktop and mobile.

### Techstack:
- HTML & CSS for structure and styling
- Vanilla JavaScript for menu filtering and search functionality
- Static site (no backend)
- Netlify for deployment

This project was my first time using AI assistance to build a static multi-page website from scratch. Before, I was completely unfamiliar with utilizing AI in code. But, this project taught me a lot about the technology. With Claude doing a lot of the coding, it gave me more time to fine tune the site's visuals. I decided the site's layout, colors, and content. I tweaked any aspect that looked even slightly off. Going forward, I would try to integrate a backend to support features like online ordering and a live inventory system. That way, the site could function as a real business tool rather than just a static mockup.
